# CASST

## Descrição

Este repositório contém o código-fonte para o site de CASST. O site foi projetado para fornecer informações sobre Soluções em Segurança do Trabalho.

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript


## Instalação e Configuração

1. Clone este repositório para seu ambiente local usando `git clone [URL do repositório]`.
2. Navegue até a pasta do projeto.
3. Abra o arquivo `index.html` em seu navegador.
